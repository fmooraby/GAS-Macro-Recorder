This Google Apps Script Macro allows the user to record macro.
For security reasons, the Macro code is embedded in the first cell of sheet 'RECORDER_OUTPUT'
If you want to run the macro, copy cell A1 of RECORDER_OUTPUT to a Macro sheet:
GO TO TOOLS -> SCRIPT EDITOR

Paste in the available script editor or in a new script file (File -> New -> Script file)
